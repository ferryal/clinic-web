| Dependency |   Version   |
| ---------- | :---------: |
| Node       | >= v18.18.2 |

## Quick Start

i) Installation

```bash
# clone git repo into `clinic-web` folder
git clone https://github.com/ferryal/clinic-web clinic-web

# install project dependencies
cd clinic-web && pnpm install
```

ii) Running app

For **development**:

```bash
pnpm run dev
```

For **production**:

```bash
# build and serve our server
pnpm run build && pnpm start
```
